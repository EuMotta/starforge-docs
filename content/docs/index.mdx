# Star Forge: Componentes reutilizáveis

A **Star Forge** é uma coleção de componentes de interface de usuário que simplifica o desenvolvimento de aplicações e acessíveis. Inspirada no shadcn/ui, o Star Forge oferece uma abordagem única: em vez de bibliotecas encapsuladas, você copia e cola código puro, mantendo total controle sobre personalização e integração. Construída com **Tailwind CSS** e **Radix UI**, ela combina design, acessibilidade e flexibilidade para desenvolvedores de todos os níveis.

<Callout title="Por que Star Forge?">
Crie interfaces rapidamente sem sacrificar controle. Com o Star Forge, você molda componentes para se alinharem perfeitamente ao seu sistema de design, com código limpo e pronto para produção.
</Callout>

## O que é o Star Forge?

O Star Forge é mais do que uma biblioteca de componentes — é uma filosofia de desenvolvimento que valoriza liberdade e simplicidade. Nossos componentes são:

- **Reutilizáveis**: Copie e cole diretamente no seu projeto, sem dependências desnecessárias.
- **Customizáveis**: Adapte estilos e comportamentos com Tailwind CSS ou edite o código TypeScript/React diretamente.
- **Acessíveis**: Construídos com Radix UI, garantindo conformidade com padrões ARIA.
- **Prontos para IA**: Código estruturado e legível, ideal para integração com ferramentas de inteligência artificial.

Seja você um desenvolvedor construindo uma Startup, um designer experimentando layouts ou uma agência buscando consistência, o Star Forge é seu aliado para desenvolver sua interface.

## Destaques

- **Integração Simples**: Adicione componentes via CLI com `npx shadcn@latest add [componente]` ou copie e cole manualmente.
- **Flexibilidade Total**: Modifique cada componente para refletir sua visão de design, sem camadas de abstração.
- **Comunidade Aberta**: Contribua com novos componentes ou melhorias no [repositório GitHub](https://github.com/EuMotta/starforge-docs).

**Exemplo: Usando o componente Button**

```tsx
import { Button } from "@/components/ui/button";

export default function Home() {
  return <Button variant="primary">Comece Agora</Button>;
}
```
