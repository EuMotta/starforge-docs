### Como usar com React Hook Form

<Steps>
  <Step>
    **Instale as dependências**
    Certifique-se de ter o `react-hook-form` instalado:
    ```bash
    npm install react-hook-form
    ```
  </Step>
  <Step>
    **Configure o formulário**
    Importe o `useForm` do `react-hook-form` e configure o componente de seleção com o `Controller` para gerenciar o estado.
  </Step>
  <Step>
    **Integre o componente**
    Passe o `control` e o `name` para o componente de seleção, conforme mostrado no código do exemplo `select-2`.
  </Step>
</Steps>

<Callout title="Nota">
  O componente com `react-hook-form` requer a configuração correta do `Controller` para funcionar adequadamente. Consulte a [documentação oficial do React Hook Form](https://react-hook-form.com/) para mais detalhes.
</Callout>
