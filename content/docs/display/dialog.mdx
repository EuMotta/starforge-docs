### Included Dialogs

- **Terms of Service** — Scrollable legal text with an accept action.
- **Login** — Email and password form.
- **Confirmation** — Generic confirm/cancel action.
- **Delete Project** — Destructive alert dialog for irreversible actions.
- **Subscribe** — Newsletter email capture.
- **Share** — Read-only project link with copy button.
- **Feedback** — Textarea for user feedback.
- **Settings** — Toggles for notifications and public profile.
- **Payment** — Credit card form with name, number, expiry and CVC.
- **Welcome** — Onboarding steps overview.
